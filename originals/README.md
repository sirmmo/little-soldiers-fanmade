## Italian glossary (from the cards)

- **Soldatino** = Little Soldier
- **Fanteria** = Infantry
- **Attacco** = Attack
- **Movimento** = Move / Movement
- **Ricognizione** = Scout (the Infantry's Special Action)
- **Avversari** = Opponents
- **Effettua** = Make / Perform
- **Posiziona** = Place
- **Rimuovi** = Remove
